# Como usar

- O relógio de cima é um temporizador que marca o tempo restante até o fim da reunião;
  - Digite nele um horário, no formato de 24 horas.
- O relógio de baixo é um temporizador que marca o tempo de fala;
  - Digite nele um valor em minutos.
- Aperte o botão na tela, **Enter** ou **Espaço** para iniciar os temporizadores;
  - Ao chegar na metade do tempo ele mostra a mensagem "metade";
  - Faltando 30 segundos ele mostra a mensagem "pra concluir";
  - Chegando em 0, o temporizador grande vira um cronômetro e mostra a mensagem "conclua".
- Aperte o botão na tela para resetar o relógio do tempo de fala.
