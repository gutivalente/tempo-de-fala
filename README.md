# Como usar

- Digite um valor em minutos e aperte **Enter** ou **Espaço** para iniciar o temporizador;
  - Ao chegar na metade do tempo ele mostra a mensagem "metade";
  - Faltando 30 segundos ele mostra a mensagem "pra concluir";
  - Chegando em 0, o temporizador vira um cronômetro e mostra a mensagem "conclua".
- Aperte o botão na tela para reiniciar o relógio.
